Beep-boop will return a range of numbers from 0 to the entered number with the following exceptions:

  1. Numbers that contain a 0 are replaced (all digits) with "Beep!"
  2. Numbers that contain a 1 are replaced (all digits) with "Boop!"
  3. Numbers divisible by 3 are replaced with "I'm sorry, Dave. I'm afraid I can't do that."

These exceptions are written from least to most important. The first exception will apply unless the second exception does, and the same with the second and third. So, for example, if your number is 10, Beep-boop will return "Boop!", and the number 30 should be replaced with "I'm sorry, Dave. I'm afraid I can't do that."
